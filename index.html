안전재고관리 및 적정구매예산 제안 서비스

재고 관리를 효율적으로 하고 적정 구매예산을 예측하는 웹 애플리케이션

프로젝트 소개

이 프로젝트는 기업의 안전재고를 관리하고 적정구매예산을 제안하는 웹 기반 서비스입니다. 머신러닝을 활용한 가격 예측과 OpenAI API를 이용한 구매예산 추천 기능을 제공합니다.

주요 기능

안전재고 모니터링

안전재고 미만인 품목을 자동으로 식별하고 대시보드에 표시합니다. 품목별 리드타임을 고려하여 재고 상태를 "재고 부족"과 "주의" 상태로 구분합니다.

구매단가 추이 및 예측

과거 구매단가 추이를 그래프로 시각화하고, 머신러닝을 통해 예측된 미래 구매단가를 제공합니다.

적정구매예산 추천

예측 구매단가와 시장 동향을 분석하여 적정구매예산을 추천합니다. OpenAI API를 활용하여 구매 전문가의 조언을 제공합니다.

구매의뢰서 작성

필요한 품목에 대한 구매의뢰서를 작성하고 제출할 수 있습니다. 수량과 단가에 따른 총 구매예산을 자동으로 계산합니다.

기술 스택

  • 백엔드: Python, Flask
  • 데이터 처리: Pandas, NumPy
  • 데이터 시각화: Plotly
  • AI 통합: OpenAI API
  • 프론트엔드: HTML, CSS, JavaScript, jQuery

설치 및 실행 방법

  1. 저장소 클론: git clone https://github.com/pamtelo/zerothon_proto.git
  2. 필요한 패키지 설치: pip install -r requirements.txt
  3. 애플리케이션 실행: python app.py
  4. 웹 브라우저에서 http://localhost:5000 접속
